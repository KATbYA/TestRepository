# Учет проектов в ИТ-компании

## Описание проекта
Программное приложение «Учет проектов в ИТ-компании» предназначено для автоматизации процессов учета и управления проектами внутри организации.  
Система позволяет хранить информацию о сотрудниках, проектах и задачах, вести контроль их выполнения, а также формировать отчеты.  

Цель разработки — повышение эффективности работы отдела разработки и прозрачности распределения задач между сотрудниками за счет внедрения информационной системы.

---

## Основные функции
- Регистрация и авторизация пользователей с различными правами доступа (администратор, руководитель, сотрудник).  
- Ведение базы данных проектов, задач и сотрудников.  
- Добавление, редактирование и удаление записей о проектах, задачах, должностях и сотрудниках.  
- Просмотр детальной информации о проекте и текущем статусе задач.  
- Формирование отчетов по выполненным и текущим задачам.  
- Просмотр сведений «О программе» (версия, разработчик, назначение).

## Быстрый старт
1. Установить Microsoft SQL Server и создать базу данных согласно SQL-скрипту.  
2. Открыть проект в Microsoft Visual Studio.  
3. В файле конфигурации (`App.config`) указать строку подключения к вашей базе данных.  
4. Собрать проект и запустить приложение (F5).  
5. Авторизоваться под учетной записью администратора для первоначальной настройки.  

---

## Структура базы данных
Приложение взаимодействует с базой данных, содержащей следующие основные таблицы:
- Projects — сведения о проектах (название, сроки, статус).  
- Tasks — список задач с указанием ответственных и сроков.  
- Employees — данные о сотрудниках компании.  
- Positions — должности и уровень доступа пользователей.  
- Users — учетные записи для входа в систему.  

---

## Роли пользователей
- Администратор — управление пользователями.  
- Руководитель — создание и распределение задач, контроль исполнения.  
- Сотрудник — просмотр и выполнение назначенных задач.
